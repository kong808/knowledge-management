Basic commands git init-ial documentation
  • git init //inicializa un nuevo repositorio local de 0
  • git remote add origin https://github.com/ivuch/myNewRepo //setea el repositorio remoto
  • git remote -v //verify the new remote URL
  • git add
  • git commit -m""
  • git push origin master #Pushes the changes in your local repository up to the remote repository you specified as the origin
  • git checkout -b myNewBranchName //to create a new branch
  • git push --set-upstream origin myNewBranchName //to set remote upstream to the new branch

  • Search Results To remove the last commit from git, you can simply run git reset --hard HEAD^ If you are removing multiple commits from the top, you can run git reset --hard HEAD~2 to remove the last two commits. You can increase the number to remove even more commits.
