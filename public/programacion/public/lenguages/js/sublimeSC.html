Sublime Text Shortcuts:

  • Ctrl+Shift+P --> Open Shortcuts Console
  • Alt+Shift+W --> Wrap element

    Macros:

  1. Ctrl+Q inicia el grabado de un Macro.
  2. Se escribe lo que se quiere grabar.
  3. Ctrl+Q (nuevamente) concluye el grabado.
  4. Se guarda desde Tools-->Save macro... Con un *nombreDescriptivo*.sublime-macro en la carpeta: SublimeText2/Packages/User

    Console Shortcuts:

  • SSHTML --> set syntax HTML (SSCSS, SSGROOVY, etc.)
